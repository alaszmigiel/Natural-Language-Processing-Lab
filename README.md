# Natural Language Processing Lab

This repo contains a collection of laboratory assignments completed during an NLP course at MUT.

## Overview
Notebooks cover various topics in Natural Language Processing and Machine Learning, including:
- text preprocessing and analysis
- TF-IDF and similarity measures
- LSA / SVD
- word embeddings
- sentiment analysis (VADER, LSTM, BERT)
- transformer-based models (Hugging Face)

## Technologies
- Python
- scikit-learn
- SpaCy / NLTK
- TensorFlow / Keras
- Hugging Face Transformers
- matplotlib / seaborn
